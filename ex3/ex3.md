## ex3　列数の違うcsvを並び替えて辞書にする。
1. input：ex2_input.csv
2. outiput：辞書型（print）で確認
3. 処理内容：列が１列の行は辞書のキーとして、その後５行分をキーの値として配列にする。
4. 例
- input.csv：  
1,  
x1_1,y1_1  
x1_2,y1_2  
x1_3,y1_3  
x1_4,y1_4  
x1_5,y1_5  
2,  
x2_1,y2_1  
x2_2,y2_2  
x2_3,y2_3  
x2_4,y2_4  
x2_5,y2_5  
・  
・  
  
- print  
```shell
{1:[[x1_1,y1_1],[x1_2,y1_2],[x1_3,y1_3],[x1_4,y1_4],[x1_5,y1_5}]],2:[[x2_1,y2_1]・・・]}･･･}
```
